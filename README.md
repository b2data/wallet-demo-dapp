# Wallet Demo Dapp

## Tonconnect Features 

- Connect
- Disconnect
- Sign Transaction
- Sign Data

## How to start locally

1. Create `.npmrc` file and set your own `_authToken`

```
@b2data:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=
```

2. Install dependencies `yarn install`

3. Run `yarn demo:start`
